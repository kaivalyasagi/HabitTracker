A full-stack habit tracking application designed to help users build consistency through daily habit monitoring and simple progress tracking.

---

## Tech Stack

- Frontend: React (Vite), Tailwind CSS  

- Backend: FastAPI (Python)  

- Database: SQLite  

---

## Features

- User authentication (login/signup)  

- Create, update, and track habits  

- Daily check-in system  

- Basic progress tracking  

- Clean and responsive UI  

---

## System Overview

The application follows a client-server architecture:

- Frontend handles user interaction and UI rendering  

- Backend manages APIs, authentication, and data logic  

- SQLite is used for lightweight persistent storage  

---

## Project Purpose

This project was built to understand full-stack development, API integration, and how backend and frontend systems interact in a real-world application.

## Setup

### Backend

cd backend  

python -m venv venv  

source venv/bin/activate   # Windows: venv\Scripts\activate  

pip install -r requirements.txt  

uvicorn main:app --reload  

### Frontend

cd frontend  

npm install  

npm run dev  

---

## Usage

1. Open the live app or run locally  

2. Sign up or log in  

3. Create habits  

4. Track daily progress  

---

## Notes

Ensure the backend server is running before starting the frontend (for local setup).
